# Nego — Talent Marketplace

Nego is a premium, dark-themed marketplace that connects clients with talent for bookable services. Clients browse profiles, book and pay with coins, message talent directly, and leave reviews. Talent manage their availability, services, and earnings, and admins oversee the whole platform.

## Who it's for

- **Clients** — browse talent, book services, send gift coins, chat in real time, and unlock premium content.
- **Talent** — manage services and pricing, accept or decline bookings, upload media, track earnings, and request withdrawals.
- **Admins** — verify identities, process withdrawals, moderate content, and monitor platform activity.

## Key features

- Real-time chat between clients and talent
- Coin-based wallet, gifting, and escrow-backed bookings
- Reviews and ratings
- In-app, push, and email notifications, with per-channel preferences
- Identity verification for clients and talent
- Admin dashboard for platform oversight

## Getting started

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000). You'll need a Supabase project and API keys for Resend (email), Paystack (payments), and Cloudinary (media uploads) configured in a local `.env` file to run the full app.

## Tech stack

Next.js, Supabase, Tailwind CSS.
